# Animal-sanctuary-for-cse370-summer2025
A DBMS based website ffor an animal sanctuary



### Running the program
1. Copy animal_sanctuary into [C:\xampp\htdocs]
2. open Xampp control panel and start Apache and Mysql
3. Go to [http://localhost/phpmyadmin/]
4. Create new database and import database [animal_sanctuary.sql].
5. Go to [http://localhost/animal_sanctuary/login.php]

### Features 
1. Breeding management and birth registration
2. Offspring tracking and health management
3. Inventory management
4. Staff user management
5. Staff shift and task management
6. Separate staff and manager dashboard
7. Animal management
8. Animal profile and maintenance
9. Enclosure management
10. User authentication for security
